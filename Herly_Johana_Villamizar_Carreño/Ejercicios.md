🧠EJERCICIOS FINALES✏️

Este archivo contiene una serie de 20 ejercicios de realizados en Python con la herramienta Google Colab enfocados en la práctica de conceptos de programación orientada a objetos.
Incluye implementaciones como operaciones de multiplicación utilizando clases, así como el uso de decoradores para modificar o extender el comportamiento de métodos.
El objetivo es reforzar principios clave como la encapsulación, reutilización de código y organización estructurada dentro del paradigma orientado a objetos.
